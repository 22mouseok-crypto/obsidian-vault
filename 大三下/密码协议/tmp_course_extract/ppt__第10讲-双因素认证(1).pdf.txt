===== PAGE 33 =====
Attacking Wang’s PSCAV
p Our attack on Hsieh-Leu’s scheme can be generalized
as follows:
(1) Intercept some transcripts during the previous login session(s);
(2) Obtain the security parameters stored in the victim’s smart card;
(3) Repeatedly guess the victim’s password via an offline automated
program.
p The following attack on Wang’s PSCAV using a
new strategy:
(1) Obtain the security parameters stored in the victim’s smart card;
(2) Return the exposed card to the victim without detection;
(3) Impersonate the server to interact with the victim (and intercept the
exchanged messages during this course);
(4) Repeatedly guess the victim’s password via an offline automated
program.

Conclusion

to USB-alike devices in two-factor schemes.

already given rise to the strongest adversary
model so far —— Type III-r.
