一些特殊的集合符号，使用 \\mathbb 命令 ：

| 名称   | 代码                         | 效果  | 说明                |
| ---- | -------------------------- | --- | ----------------- |
| 实数集  | \\mathbb{R} 或 \\mathbb R   |     | real numbers      |
| 复数集  | \\mathbb{C}                |     | complex numbers   |
| 整数集  | \\mathbb{Z}                |     | integers          |
| 自然数集 | \\mathbb{N}                |     | natural numbers   |
| 正整数集 | \\mathbb{N^+}或 \\mathbb{W} |     | whole numbers     |
| 质数集  | \\mathbb{P}                |     | prime numbers     |
| 有理数集 | \\mathbb{Q}                |     | rational numbers  |
| 无理数集 | \\mathbb{I}                |     | irational numbers |
| 空集   | \\emptyset                 |     |                   |
